# 펭귄 데이터셋 개요

Palmer Penguins 데이터셋의 탐색적 분석 결과를 요약합니다. 각 슬라이드에는 시각화와 핵심 인사이트가 포함되어 있습니다.

::: notes
데이터 출처와 분석 목적(종 식별, 형태학적 차이 파악)을 간단히 소개하세요.
:::

# 데이터 샘플 (상위 10행)

데이터의 상위 10행을 확인합니다. (원본 표는 보고서 참조)

::: notes
컬럼 설명: `bill_length_mm`, `bill_depth_mm`, `flipper_length_mm`, `body_mass_g`, `species`, `island`, `sex`.
:::

# 기술 통계 요약

숫자형 변수의 평균, 표준편차, 사분위수 등을 요약합니다. (자세한 표는 보고서 참조)

::: notes
`body_mass_g` 평균 약 4200g, `flipper_length_mm` 평균 약 201mm를 언급.
:::

# 결측치 요약

결측치 현황: `bill_length_mm`:2, `bill_depth_mm`:2, `flipper_length_mm`:2, `body_mass_g`:2, `sex`:11

::: notes
결측치는 특정 시각화에서 `dropna()`로 처리했음을 공지하세요.
:::

# 부리 길이 vs 부리 깊이 (종별)

![](output/scatter_bill.png)

**인사이트:**

부리 길이와 부리 깊이의 산점도는 종별로 명확한 군집을 형성합니다. Gentoo는 상대적으로 긴 부리와 얕은 깊이의 패턴을 보이며, Adelie는 중간 범위, Chinstrap는 다른 조합을 보입니다. 이러한 형태적 차이는 먹이 섭취 방식과 생태적 적응을 반영할 가능성이 큽니다. 아웃라이어와 일부 결측값을 고려하면, 회귀분석과 군집분석을 통해 각 종의 형태적 특징을 정량화할 수 있습니다.

::: notes
포인트: 군집 예시(좌표 범위), 후속 분석 제안(클러스터링, 회귀).
:::

# 체중 히스토그램

![](output/hist_body_mass.png)

**인사이트:**

체중 분포는 종 간 중심위치 차이를 보입니다. Gentoo는 고체중대, Adelie와 Chinstrap는 상대적으로 낮거나 중간 분포를 갖습니다. 분포의 꼬리나 뾰족함은 표본 편향 혹은 실제 개체군의 군집성을 반영할 수 있으며, 필요시 로그 변환으로 분포 정규화 후 분석을 권장합니다.

::: notes
언급: 이상치 검토, 분포 정규화(로그) 제안.
:::

# 체중 KDE (종별)

![](output/kde_body_mass.png)

**인사이트:**

KDE는 종별 체중의 분포 중첩과 분리를 부드럽게 보여줍니다. 분포가 분리된 구간은 단일 변수만으로도 종 구분에 기여할 수 있음을 시사하며, 겹침이 큰 구간은 다변량 접근이 필요함을 의미합니다. 또한 분포 폭이 넓은 종은 개체군 내 변이가 크다는 신호입니다.

::: notes
밴드폭 선택의 영향과 분포 겹침의 해석을 간단히 설명.
:::

# 체중 상자그림 (종별)

::: notes
권장: 통계검정과 이상치 원인 조사.
